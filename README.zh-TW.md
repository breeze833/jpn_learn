# 🇯🇵 日語學習助手 (Japanese Learning Assistant)

**繁體中文** | [English](README.md)

這是一個專為日語初學者設計的網頁工具。**免安裝、免註冊、免架設伺服器**，只有一個檔案，打開瀏覽器即可立刻開始練習。

無論是剛開始背誦 50 音的新手，還是想要練習朗讀句子的進階學習者，都能透過這個工具獲得幫助。

## 🚀 30秒快速嘗鮮 (Quick Start)

您可以直接下載單一 HTML 檔案，無需任何安裝即可使用：

1.  **取得檔案**：[📥 點擊此處前往 jpn_learn.html 下載頁面](jpn_learn_app/jpn_learn.html)
    * *進入頁面後，請點擊右上角的 <kbd>⬇</kbd> (Download raw file) 按鈕。*
2.  **直接開啟**：雙擊下載好的檔案，或拖曳至瀏覽器中。
3.  **開始使用**：程式載入後即可開始學習！

> **注意**：由於程式需要載入外部字典檔與語音引擎，請確保您的裝置目前**已連上網路**。

## 🎯 主要功能

### 1. 互動式 50 音練習卡
不再死記硬背！透過視覺與聽覺的雙重輔助，幫您快速記憶平假名與片假名。

* **平/片假名切換**：一鍵快速切換，對照學習。
* **筆順動畫**：點擊卡片即可查看正確的書寫筆順。
* **記憶口訣**：每個字母都附帶有趣的「聯想記憶法」（例如：「あ」長得像「安」全的安），讓形狀與發音瞬間連結。
* **真人/合成發音**：點擊卡片即可聆聽標準讀音。

### 2. 智慧句子解析器 (讀音標註)
看到一長串日文句子不知道怎麼唸嗎？貼上來就對了！

* **自動斷詞與標音**：將句子拆解成單字，並自動標上平假名讀音 (Furigana)。
* **逐詞朗讀模式**：系統會像「卡拉 OK」一樣，唸到哪個字就亮起哪個字，非常適合練習跟讀 (Shadowing)。
* **單字發音**：滑鼠停留在個別單字上，可以單獨聆聽該單字的發音。

### 3. 個人化語音設定
* **語速控制**：可自由調整語速 (0.5x ~ 2.0x)。
* **雙引擎切換**：支援「本地語音 (離線可用)」與「遠端語音 (Google 高品質)」，系統會自動偵測最佳選項。

## 📖 使用教學

### 如何開始 50 音練習？
1.  點擊上方的 **「50音練習」** 分頁。
2.  選擇 **「平假名」** 或 **「片假名」** 模式。
3.  點擊任意字母卡片，查看詳細筆順與口訣。

### 如何解析句子？
1.  點擊上方的 **「句子解析」** 分頁。
2.  在輸入框內貼上日文句子（例如：`今日の天気はとてもいいですね`）。
3.  按下藍色的 **「🔍 解析」** 按鈕。
4.  解析完成後，可選擇 **「🔊 整句」** 朗讀或 **「▶️ 逐詞」** 跟讀。

## 🤖 關於本專案的誕生

這是一個實驗性質的專案，展示了人與 AI 協作的新模式。([閱讀開發故事](https://medium.com/@breeze833/與-ai-協作寫程式-我如何打造一個-會說話-的日語學習助手-8865258703ac))

本程式的**每一行程式碼**，皆是透過與 **Google Gemini 3** 的「純對話」過程中產生。開發過程中並未依賴專業的 AI 寫程式介面（如 Cursor 或 GitHub Copilot），而是就像聊天一樣，由人類提出構想、需求、與高階方案，AI 擔任架構師與工程師，共同一步步除錯、優化並實現功能。

這證明了即使不依賴複雜的開發環境，透過自然語言的溝通，也能創造出實用且完整的應用程式。
